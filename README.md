## 📦 How to Run Locally

To preview the portfolio website on your local machine:

1. Clone this repository:
   ```bash
   git clone https://github.com/ReshuUpadhyay231/portfolio.git
   cd portfolio
   ```
2. Start a local server:
   * **Python**:
     ```bash
     python -m http.server 8000 --bind 127.0.0.1
     ```
   * **Node/NPM** (using live-server):
     ```bash
     npx live-server
     ```
3. Open `http://localhost:8000` on your web browser.
